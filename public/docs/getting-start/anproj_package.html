package.jsonの作成方法

package.jsonの作成方法に関する資料です。

はじめに

package.jsonはゲームの初期設定を決定する重要な役割を果たしています。

package.jsonはanprojファイル正式採用となるv0.9.0-rc1以降で利用可能です。

game-name

ゲームタイトルを設定可能です。

記述: 任意

初期設定: "" (タイトルなし)

利用可能バージョン: v0.9.0-rc1~

例

{

  "game-name: "Alice wonder land"

}


first-read

初回ロード時に読み込むファイルを設定できます。

記述: ?

初期設定: ?

利用可能バージョン: v0.9.0-rc1~

例

{

  "first-read": "story/main.anov"

}
