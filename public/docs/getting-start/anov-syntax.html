Anov構文

Alice Novel専用の構文

概要

anov構文とは、Alice Novelで採用している記述形式です。

もくじ

  • Hello World! を表示する
  • キャラクターに喋らせてみる
  • 画像ファイルの使い方 (記述予定)
  • 音声ファイルの使い方 (記述予定)
  • 動画ファイルの使い方 (記述予定)

Hello World! を表示する

Alice Novelは、ノベル系のゲームを想定して作成されているため、喋り手の名称を表示する場所があります。(名前部分)

また、その喋り手が喋っている内容を表示する場所があります。(セリフ部分)

ここでは手始めにセリフ部分に「Hello World!」と表示する構文を書いてみましょう。

[Hello World!]

このように、[ ]で囲むことで、文字を表示することが可能です。

キャラクターに喋らせてみる

先述の通り、Alice Novelは、キャラクターの名前を表示することが可能です。

- キャラクター1

[こんにちは!]

このように- に続けて名前を書くことでキャラクターの名前を表示できます。
