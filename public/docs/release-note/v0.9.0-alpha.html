Alice Novel Docs

v0.9.0-alpha版

v0.9.x系では初のリリース。

概要

旧NEO_Novelから、正式な開発を進めるために始まった計画。

まずは、初歩的な機能のみの実装を目標として開発を開始していく予定。

新機能

  • ファイル読み込み機能

    • (1行ごとだが、読み込み自体は可能。)

  • キャラクター名前欄/セリフの反映機能

    • 読み込んだファイルからキャラクターの名前とセリフを反映可能に。

    ただし、画像系は現時点では非対応。

破壊的変更点

今回が実質初リリースのため、特になし。
